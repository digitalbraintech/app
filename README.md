# digitalbrain_flutter

> **Destination repos:** this folder splits across two `digitalbrain/*` repos at extraction time:
> - 🔓 `digitalbrain/UIKit` (pub.dev `digital_brain_ui` package) — native-widget wrappers (markdraw canvas, video, future widgets)
> - 🔒 `digitalbrain/UI` — the polished 3D brain renderer, BrandWordmark, chrome, themes, gRPC client, RFW host wiring
>
> See [`docs/strategy/repo-destination-map.md`](../../docs/strategy/repo-destination-map.md) and [§4 of the monetization design spec](../../docs/superpowers/specs/2026-05-20-experience-bundle-monetization-design.md).
